### 변수

변수 작성 규칙은 다음과 같다. 주의사항은 주석에!

출력은 파이썬과 비슷하게 print를 쓰면 된다.


```swift

// 1. var 변수명 = 값
// 2. var 변수명 : 타입 = 값
// 띄어쓰기 준수, 타입 대문자 준수
// int(x) , Int(o)

var v1 = 123
var v2 : Int = 123

print(v1)
print(v2)

```
