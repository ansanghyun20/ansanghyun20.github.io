### 변수

변수 작성 규칙은 다음과 같다.

변수를 작성할 때 띄어쓰기를 정확히 준수해야하며 타입에 적을 문자의 대,소 문자 형식이 정확하게 들어 맞아야 한다.


```swift

// 1. var 변수명 = 값
// 2. var 변수명 : 타입 = 값
// 띄어쓰기 준수, 타입 대문자 준수
// int(x) , Int(o)

var v1 = 123
var v2 : Int = 123

print(v1)
print(v2)

```
